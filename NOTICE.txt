Lista de Materiais
==================

Dependencies
------------

Lista de Materiais requires the following software to run:

 * Java SE JRE >= 1.5
   URL: http://java.sun.com/
   
 * SLF4J
   URL: http://www.slf4j.org/

Lista de Materiais requires the additional software to compile:

 * Java SE SDK >= 1.5
   URL: http://java.sun.com/

 * Ant
   URL: http://ant.apache.org/

 * Maven
   URL: http://ant.apache.org/

 * JUnit >= 4.11
   Notes: For the tests only.
   URL: http://www.junit.org/
